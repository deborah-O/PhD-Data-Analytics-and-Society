# Model Output Figures

This directory contains figures generated by the **model analysis and inference pipeline** used in the manuscript.

All figures in this folder are **reproducible outputs** produced by the model simulation and Approximate Bayesian Computation (ABC) inference workflow. These images correspond to figures referenced in the **main body of the manuscript**.

The figures are generated automatically when running the repository's analysis pipeline.

## Reproducibility

These figures are generated from the simulation and inference pipeline implemented in the repository.

To reproduce the results:

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the analysis pipeline

```
bash scripts/run_pipeline.sh
```

The pipeline will regenerate the data and output figures contained in this directory.
